What is Birdfingers?

Birdfingers is a local web UI to manage Python packages in virtual environments and Windows embedded Python. It supports listing/uninstalling packages, installing exact versions from PyPI, and snapshotting environments for preview/restore/diff.

Run

python birdfingers_pkgmgr.py
# opens http://127.0.0.1:8765/

Use --port to change port. Activate your venv first to target it.

Embedded Python

  1. Click Enable site-packages to add import site to pythonXY._pth.
  2. Click Check / install pip to verify or bootstrap pip.

  • Uninstall ‚Üí prompts, then removes the package

Install Tab

  • Install latest or Install selected
  • Accepts package==x.y.z in the name field

Snapshots Tab

  • Save snapshot with optional name/comment
  • Preview shows install/uninstall plan vs current
  • Diff A‚ÜîB between two snapshots
  • Restore runs pip install -r snapshot requirements

Bottom Bar (Actions & Output)

  • Chips: click to set **active** package; √ó to remove
  • Buttons: **Show details**, **Uninstall selected**, **Load versions**, **Install version**
  • Output: live pip logs with job id/state

Files & Logs

  • Set BIRDFINGERS_DATA_DIR to choose storage location
  • Logs: birdfingers.log (text), birdfingers_audit.jsonl (JSONL)
  • Snapshots folder: birdfingers_snapshots/
